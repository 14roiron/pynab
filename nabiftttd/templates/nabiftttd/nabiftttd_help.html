{% trans "How to create an IFTTT applet that works with your Nabaztag" %}

Step 1 : Create an IFTTT acount

Just go there and follow the procedure. You can select a free account, it's working perfectly.

Step 2 : Create an Applet with Webhooks

The trigger event will be a webhook, this means that when Nabaztag sees a specific NFC tag it will call IFTTT to tell you this good news and based on this you'll be able to trigger diffent actions (there are dozens diff√©rent actions available on IFFFT). Go to : https://ifttt.com/create

Create an applet

Choose a service, serach for Webhooks

Select "Receive a Web request with a JSON payload"

Type an event name. This event name will be use when you program your NFC tag (here it's "ExampleOfAnEventName")

You need now to configure an action

As an example, let's choose email

Create action then Continue

And finish and you've just created an applet'

Step 3 : Configure the service here on Nabaztag

Go to : http://nabaztag.local/services/

And fill out the following screen with the key of the Webhooks service.

You'll only have to do this once and the key is here : https://ifttt.com/maker_webhooks and click on Documentation.

Step 4 : Configure your NFC tag

Go to http://nabaztag.local/rfid/to program your tag

You need to enter the same event name as the one you used in IFTTT.

And you're done !
